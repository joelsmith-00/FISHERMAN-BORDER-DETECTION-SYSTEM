# Fisherman Border Detection

Marine Guardian AI is a static web dashboard prototype focused on coastal safety for small-scale fishermen.

## Features

- AI-oriented maritime safety presentation UI
- Border-risk and telemetry visualization components
- Emergency communication concept (SOS mesh network)
- Responsive sections for identity, problem, features, technical components, and data flow

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)
- Tailwind CSS (CDN in `index.html`)

## Project Structure

```text
.
├── app.js
├── index.html
├── styles.css
└── assets/
```
